## How to run

1. Clone the repository.

```sh
git clone https://github.com/software-engineering-2023/Code-PROs.git
```

2. Move to the website's directory.

```sh
cd Code-PROs/Online-Bank-Website
```

3. Open Home Page and start navigating.

```sh
start index.html
```
